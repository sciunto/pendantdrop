Install
=======

Via pip
-------

pendantdrop is availlable on `pypi <https://pypi.org/project/pendantdrop/>`__.

Run

    pip install pendantdrop


Note about dependencies
-----------------------

* numpy>=1.15.1
* scipy>=1.4.0
* pandas>=0.23.0
* matplotlib>=3.0.1
* scikit-image>=0.16
* joblib>=0.15.1
* appdirs>=1.4.3
